## 构建 - 创新 - 扩展

SystemForge Studio (SFS) 是一个现代软件工程工作室，专注于使用清晰架构和现代开发实践，构建可扩展的 Web、移动端、后端以及云就绪系统。

这个作品集网站代表 SFS 面向公众的工程形象，展示我们的服务、技术能力、架构经验和软件交付方式。

# 重点项目

## EquiTip 现代化

将旧有 low-code 移动平台迁移为可扩展的 React Native 与微服务架构，使用 PostgreSQL、API Gateway、JWT 身份验证和安全后端服务。

## AI 防坠落平台

移动安全平台，包含 AI 辅助的危险检测、引导式工作流、离线优先同步以及 PDF 导出能力。

## 定制业务系统

为业务运营开发可扩展的内部工具、仪表板、API、数据库和工作流自动化系统。

# 开发设置

克隆仓库：

```bash
git clone https://github.com/systemforge-studio/systemforge-website.git
```

进入项目目录：

```bash
cd systemforge-website
```

安装依赖：

```bash
npm install
```

运行开发服务器：

```bash
npm run dev
```

构建生产版本：

```bash
npm run build
```
