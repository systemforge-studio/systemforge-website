## 打造 - 創新 - 擴展

SystemForge Studio (SFS) 是一個現代軟體工程工作室，專注於運用清晰架構與現代開發實務，打造可擴展的 Web、行動端、後端與雲端就緒系統。

這個作品集網站代表 SFS 對外的工程形象，展示我們的服務、技術能力、架構經驗與軟體交付方式。

---

# 關於

SystemForge Studio 協助企業將想法轉化為可擴展、安全、可上線營運的軟體解決方案。

我們的工程方法重視：

- 清晰架構
- 易維護系統
- 可擴展後端設計
- 安全 API
- 行動端與 Web 平台
- 雲端就緒部署
- 現代工程工作流程

我們專精於全端工程、後端系統、行動端開發、系統架構、整合，以及現代部署流程。

# 精選專案

## EquiTip 現代化

將既有 low-code 行動平台遷移為可擴展的 React Native 與微服務架構，並使用 PostgreSQL、API Gateway、JWT 驗證與安全後端服務。

## AI 防墜平台

行動安全平台，包含 AI 輔助危害偵測、引導式工作流程、離線優先同步，以及 PDF 匯出能力。

## 客製化商務系統

為企業營運開發可擴展的內部工具、儀錶板、API、資料庫與工作流程自動化系統。

# 開發設定

複製儲存庫：

```bash
git clone https://github.com/systemforge-studio/systemforge-website.git
```

進入專案目錄：

```bash
cd systemforge-website
```

安裝相依套件：

```bash
npm install
```

啟動開發伺服器：

```bash
npm run dev
```

建置正式版本：

```bash
npm run build
```
