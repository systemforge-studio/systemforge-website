## دروستکردن - داهێنان - گەشەپێدان

SystemForge Studio (SFS) ستۆدیۆیەکی هاوچەرخی ئەندازیاری نەرمەواڵەیە کە گرنگی دەدات بە دروستکردنی سیستەمی وێب، مۆبایل، backend، و سیستەمە ئامادە بۆ هەور کە بتوانن گەشە بکەن، بە بەکارهێنانی clean architecture و ڕێبازە نوێکانی گەشەپێدان.

ئەم ماڵپەڕەی پۆرتفۆلیۆیە نوێنەرایەتی ئامادەبوونی گشتی ئەندازیاری SFS دەکات و خزمەتگوزارییەکان، تواناکانی تەکنیکی، ئەزموونی architecture، و ڕێبازی گەیاندنی نەرمەواڵە پیشان دەدات.

---

# دەربارە

SystemForge Studio یارمەتی کۆمپانیاکان دەدات بیرۆکەکان بگۆڕن بۆ چارەسەری نەرمەواڵەیی قابلی گەشەکردن، پارێزراو، و ئامادە بۆ بەرهەمهێنان.

ڕێبازی ئەندازیاری ئێمە گرنگی دەدات بە:

- clean architecture
- سیستەمی ئاسان بۆ چاکسازی و پاراستن
- دیزاینی backendی قابلی گەشەکردن
- APIی پارێزراو
- پلاتفۆرمی مۆبایل و وێب
- ناردنی ئامادە بۆ هەور
- workflowی ئەندازیاری هاوچەرخ

ئێمە تایبەتمەندین لە full-stack engineering، سیستەمەکانی backend، گەشەپێدانی مۆبایل، system architecture، تێکەڵکردنەکان، و deployment pipelineی هاوچەرخ.

# ئامادەکردنی گەشەپێدان

repository clone بکە:

```bash
git clone https://github.com/systemforge-studio/systemforge-website.git
```

بچۆ ناو پڕۆژەکە:

```bash
cd systemforge-website
```

dependencies دابمەزرێنە:

```bash
npm install
```

development server کارپێبکە:

```bash
npm run dev
```

وەشانی production دروست بکە:

```bash
npm run build
```
